# 規約・プライバシー運用

更新日: 2026-07-14

## 管理画面でできること

- イベント規約とプライバシーポリシーの下書き作成
- 下書きのタイトル・本文編集
- 版指定での公開
- 公開中文書と過去版の閲覧
- 公開ページの確認

## 公開手順

1. 管理トップの「規約・プライバシー」を開く。
2. 文書種類、版番号、タイトル、本文を入力して下書きを作成する。
3. 下書きの表示と内容を確認し、必要なら修正する。
4. 「公開する」を押し、確認ダイアログで確定する。
5. 公開ページを開き、スマートフォンでも最終確認する。

## 安全ルール

- 公開後の文書は編集不可。修正は新しい版を作り公開する。
- 新版の公開時に旧版は過去版になる。旧版の同意記録は上書きしない。
- 公開と同時にイベントの同意対象版が更新される。
- 設定版と一致する公開文書が2種類とも揃うまで受付開始できない。
- 監査ログには種類・版・状態のみを記録し、本文は記録しない。

## 残作業

2026-08-08用の正式文面と版番号は主催者確定待ち。確定まで仮文面を公開しない。
